# i18n AGENTS

## Propósito

Fuentes de traducción compartidas del producto.

## Relevancia

- `mobile` genera código a partir de estos archivos.
- `web` y docs también dependen de textos/traducciones en distintos puntos.

## Regla práctica

- Cualquier cambio acá puede tener efectos transversales.
- Mantener formato ordenado y consistente; el repo trata esta carpeta casi como data.
